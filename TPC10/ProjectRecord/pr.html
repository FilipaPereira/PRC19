Abstract:

Este trabalho de casa tinha como objetivo a implementação da interface relativa aos atores do ontologia de cinema que tem sido trabalhada durante as aulas. Para tal foi necessário não só criar as páginas para a lista de atores e a respetiva página individual de cada um, mas também atualizar a API de modo a fornecer os dados necessários a essas páginas.

Assim, foi criado um novo controlador e roteador para os atores, que responde a dois pedidos GET: /atores e /atores/:id. Na página individual do ator estão listados os filmes em que este participou, bem como a respetiva personagem que interpretou. Na interface, de modo a renderizar as duas novas páginas foram criadas duas views: AtorPrincipal e ConsultaAtor.

Deliverables:

  • Rota dos Atores
  • Controlador dos Atores
  • Ficheiro Vue referente à Lista dos Atores
  • Ficheiro Vue referente à Página Individual do Ator
