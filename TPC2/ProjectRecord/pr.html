Abstract:

Este trabalho de casa tinha como objetivo a criação de uma ontologia para um Arquivo Musical e a resposta a algumas queries sobres os dados presentes nesse arquivo. Assim, a ontologia definida contém 3 entidades: Obra, Instrumento e Partitura. Sendo que cada obra utiliza instrumentos, que, por sua vez, são definidos por partituras.

Após a criação da ontologia inicial com um indivíduo de cada entidade, de modo a obter o formato do mesmo em Turtle, foi aplicada uma stylesheet para transformar o ficheiro XML em Turtle. O resultado desta transformação foi adicionado à ontologia inicial. Subsequentemente, para verificar que tudo correu bem e para responder às queries, a ontologia foi importada no GraphDB. Aí utilizamos SPARQL para responder à um conjunto de queries sobre os dados.

Deliverables:

  • Ficheiro XML do arquivo musical
  • Stylesheet usada para transformar o XML em Turtle
  • Ontologia inicial
  • Ficheiro output após aplicação da stylesheet
  • Ontologia final
  • Respostas às queries
