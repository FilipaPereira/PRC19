Abstract:

Este trabalho de casa tinha como objetivo a criação de um servidor NodeJS que suporte uma interface para a execução de queries em SPARQL. Assim, é possível escolher o repositório sobre o qual será efetuada a query e escrever a mesma por texto, sendo que todos os repositórios GraphDB existentes são inicialmente carregados e o utilizador apenas escolhe o repositório pretendido.

Após inserir a query, o utilizador carrega no botão Executar, ao qual está associado uma ação que efetua a query e coloca os resultados da mesma no espaço destinado aos resultados no lado direito da página. Também é possível limpar esta área através do botão Limpar. Para não ser necessário dar refresh à página e para que esta seja mais reactiva foi utilizado AJAX nas ações associadas aos botões.
