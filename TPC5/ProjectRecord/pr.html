Abstract:

O trabalho de casa desta semana tinha como objetivo gerar um dataset de nadadores através de uma query SPARQL sobre os dados contidos na DBPedia. Este dataset apresenta os seguintes campos: Nome, Género, Data de Nascimento, Local de Nascimento, Data de Falecimento e Abstract, sendo que estes últimos três são opcionais.

Deliverables:

  • Query SPARQL para obter o dataset
  • Dataset de Nadadores
  • Project Record - XML
